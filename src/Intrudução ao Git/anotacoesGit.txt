    
    ## - Desde sua instalção e configurações tanto no Windows, Mac e o Linux. 

    ## git config Usando sua Identidade: 

    - git config --global user.name "John Doe" 
    - git config --global user.email johndoe@example.com 

    Ativando Cores: - git config --global color.ui true 

    ## git help Exitem três maneiras de ajuda

    - git help {comando} - git {comando} --help 
    - man git- {comando}

    - git commit
    - git reset
    - git rm
    - git pull
